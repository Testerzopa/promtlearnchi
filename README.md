# Chinese Vocabulary Hub (HSK 1-4)

เว็บไซต์รวบรวมคำศัพท์ภาษาจีนพื้นฐานตั้งแต่ระดับ HSK 1 ถึง HSK 4 ออกแบบมาเพื่อช่วยให้ผู้เริ่มต้นเรียนภาษาจีนสามารถทบทวนคำศัพท์ได้อย่างสะดวกรวดเร็ว

🎯 **ดูหน้าเว็บจำลองได้ที่:**
https://testerzopa.github.io/promtlearnchi/

---

## ✨ ฟีเจอร์เด่น (Features)

*   📚 **คลังคำศัพท์แบ่งระดับ:** รวบรวมและจัดหมวดหมู่คำศัพท์ตามมาตรฐาน HSK 1, HSK 2, HSK 3 และ HSK 4
*   🔍 **ข้อมูลครบถ้วน:** แสดงตัวอักษรจีน (ตัวย่อ), พินอิน (Pinyin), และคำแปลภาษาไทย
*   📱 **Responsive Design:** หน้าเว็บปรับขนาดตามอุปกรณ์อัตโนมัติ รองรับทั้งคอมพิวเตอร์ แท็บเล็ต และมือถือ

---

## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)

*   **HTML5:** โครงสร้างเว็บเว็บและการจัดเรียงเนื้อหา
*   **CSS3:** การตกแต่งหน้าตา, Responsive

## 🚀 วิธีการใช้งาน (Getting Started)

1.  **Clone โปรเจกต์นี้ไปยังเครื่องของคุณ:**
    ```bash
    git clone https://github.com/Testerzopa/promtlearnchi.git
    ```
2.  **เปิดใช้งาน:**
    *   ดับเบิ้ลคลิกที่ไฟล์ `index.html` เพื่อเปิดใช้งานบนบราวเซอร์ได้ทันที
    *   หรือใช้ Extension เช่น **Live Server** บน VS Code เพื่อดูการเปลี่ยนแปลงแบบ Real-time
